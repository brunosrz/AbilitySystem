ASEditorPlugin
==============

**Inherits:** ``EditorPlugin``

Main editor plugin for the Ability System.

.. rst-class:: classref-introduction-group

Description
-----------

**ASEditorPlugin** is the main entry point for the Ability System editor integration. It registers custom types, icons, and handles the initialization of inspector extensions and the tags panel.
