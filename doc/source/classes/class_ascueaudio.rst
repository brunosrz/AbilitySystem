ASCueAudio
==========

**Inherits:** :ref:`ASCue<class_ASCue>` **<** ``Resource``

Cue specialized for playing audio.

.. rst-class:: classref-introduction-group

Description
-----------

**ASCueAudio** is a specialized cue that automatically plays audio when triggered. It works with ``AudioStreamPlayer``, ``AudioStreamPlayer2D``, and ``AudioStreamPlayer3D`` nodes provided to the :ref:`ASComponent<class_ASComponent>`.



Create AudioStream resources in your project (.wav, .ogg, .mp3), then create a .tres resource of this type and assign the :ref:`audio_stream<class_ASCueAudio_property_audio_stream>` property. Add it to your :ref:`ASAbility<class_ASAbility>`'s :ref:`ASAbility.cues<class_ASAbility_property_cues>` list.



The system automatically tries AudioStreamPlayer first, then AudioStreamPlayer2D, then AudioStreamPlayer3D, unless a :ref:`ASCue.node_name<class_ASCue_property_node_name>` is specified to target a specific node.
