ASTagsPanel
===========

**Inherits:** ``VBoxContainer``

Editor UI for managing Ability System Tags.

.. rst-class:: classref-introduction-group

Description
-----------

**ASTagsPanel** is a custom editor component displayed within the Project Settings. It allows developers to add, remove, and list global Ability System Tags used by the Ability System.

The panel supports hierarchical organization using dot notation (e.g., ``state.stun``) and classifies tags into **Name Tags** (identity) and **Conditional Tags** (status/state).
