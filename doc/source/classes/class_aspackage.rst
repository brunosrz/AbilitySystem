ASPackage
=========

**Inherits:** ``Resource``

A reusable container for effects and cues that can be delivered.

.. rst-class:: classref-introduction-group

Description
-----------

**ASPackage** is a data container that bundles multiple :ref:`ASEffect<class_ASEffect>`\ s and :ref:`ASCue<class_ASCue>`\ s together. This is primarily used by :ref:`ASDelivery<class_ASDelivery>` to apply a complex payload (like a fireball's damage and visual effects) to a target :ref:`ASComponent<class_ASComponent>`.
