ASEffect
========

**Inherits:** ``Resource``

Resource defining modifications to attributes and tags.

.. rst-class:: classref-introduction-group

Description
-----------

**ASEffect** defines a package of attribute modifiers and tags. Can be instant, have a duration, or be infinite.
