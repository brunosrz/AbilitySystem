ASCueAnimation
==============

**Inherits:** :ref:`ASCue<class_ASCue>` **<** ``Resource``

Cue specialized for playing animations.

.. rst-class:: classref-introduction-group

Description
-----------

**ASCueAnimation** is a specialized cue that automatically plays animations when triggered. It works with ``AnimationPlayer``, ``AnimatedSprite2D``, and ``AnimatedSprite3D`` nodes provided to the :ref:`ASComponent<class_ASComponent>`.



Create a .tres resource of this type, set the :ref:`animation_name<class_ASCueAnimation_property_animation_name>` property, and add it to your :ref:`ASAbility<class_ASAbility>`'s :ref:`ASAbility.cues<class_ASAbility_property_cues>` list. When the ability is activated, the animation will play on the target node.



\ **Example:**\
