ASBridgeState
=============

**Inherits:** ``LimboState``

LimboState with integrated Ability System support.

.. rst-class:: classref-introduction-group

Description
-----------

A ``LimboState`` that integrates with Ability System. Provides automatic event dispatching on state transitions and event listening for automatic transitions.

Can require specific tags to be present before entering, and dispatches AS events on ``_enter``, ``_exit``, and ``_update``.
